# Koo Private Bar

A small mobile web app for a private home bar — guests browse the drink menu on their
phone and place an order; the host runs a live queue. Static site, served on GitHub Pages.

- **Menu** lives in [`drinks.json`](drinks.json) — edit that file to change drinks.
- **App** is a single file, [`index.html`](index.html) (vanilla HTML/CSS/JS, no build step).
- Unlisted: `robots.txt` blocks search indexing. Shareable by link only.
